Instructions

To play this game with an arduino esplora board, please download the arduino files on my github. To run the two files on different ports use the "Script Editor" on Mac to open two instances of the Arduino IDE.

Once you have connected the two different codes to the two arduinos you will be able to play the game with the esploras either by downloading and running the processing program, or by using the online version
